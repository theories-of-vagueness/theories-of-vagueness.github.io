# Formal Theories of Vagueness

Static course website for GitHub Pages.

## Files

- `index.html`: main course page
- `styles.css`: site styling
- `slides/`: upload slide PDFs here
- `exercises/`: upload exercise PDFs here

## Expected filenames

- `slides/lecture-1-slides.pdf`
- `slides/lecture-2-slides.pdf`
- `slides/lecture-3-slides.pdf`
- `slides/lecture-4-slides.pdf`
- `slides/lecture-5-slides.pdf`
- `exercises/lecture-1-exercises.pdf`
- `exercises/lecture-2-exercises.pdf`
- `exercises/lecture-3-exercises.pdf`
- `exercises/lecture-4-exercises.pdf`
- `exercises/lecture-5-exercises.pdf`

## GitHub Pages

1. Push this repository to GitHub.
2. In the repository settings, open `Pages`.
3. Set the source to `Deploy from a branch`.
4. Select your main branch and the `/ (root)` folder.

After that, the site will serve `index.html` automatically.
